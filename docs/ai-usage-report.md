# AI Usage Report

## Tools Used & Use Cases

- **GitHub Copilot**: Used for code generation, HTML/CSS/JS suggestions, and patching files directly in the workspace.
- **Copilot Chat**: Used for interactive Q&A, design advice, and step-by-step implementation guidance.

### Use Cases

- Generating initial portfolio HTML structure and sections.
- Creating initial CSS for styling and responsive design.
- Inserting images, icons, and emojis for improved user experience.

## Benefits & Challenges

### Benefits

- Accelerated development and prototyping.
- Reduced manual coding for repetitive tasks.
- Quick access to best practices and modern design patterns.

### Challenges

Ai added a lot of redundant code so I had to clean the code specially after i ask it to edit an existing feature.

## Learning Outcomes

- Improved understanding of HTML, CSS, and JavaScript integration.
- Gained experience in UI/UX design and accessibility.
- Learned how to leverage AI tools for rapid prototyping and iteration.
- Developed skills in reviewing, editing, and customizing AI-generated code.

## How I Modified AI Suggestions Responsibly

- Reviewed all AI-generated code for accuracy and relevance.
- Customized layouts, colors, and content to match personal and project needs.
- Ensured accessibility and responsiveness in design.
- Tested features and fixed issues that arose from AI suggestions.
- Used the AI-generated code as a base and improved or fixed functionality myself.
